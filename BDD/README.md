## **Project Title**

'Dashboards' is the the naming of the current project and it is the section of ReportPortal

## **Introduction**

The goal of the project is to cover by automated test 'Dashboards' section using BDD framework Cucumberf.

## **Installation**

To install Project Title, follow these steps:

1. Clone the repository: **`git clone https://github.com/yauhenFan/advanced.git`**
2. Navigate to the project directory: **BDD`**
3. Install dependencies: **`npm install`**
4. Start the project: **`npm run test`**
